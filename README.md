# geonosis-devtools

Personal reusable Claude skills, subagents, GitHub Actions workflows, 
Makefiles, and pytest conventions. Used across projects, owned by Scott.

## Structure

- `claude/skills/` — Claude Code skill files per domain
- `claude/subagents/` — reusable subagent templates
- `github-actions/` — reusable workflow templates
- `makefiles/` — Makefile patterns
- `pytest/` — pytest conventions and conftest templates
